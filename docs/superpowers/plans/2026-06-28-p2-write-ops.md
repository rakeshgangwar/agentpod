# P2 — Write Operations & Durable Terminal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the read-only facilities console into one that can act on a station — a durable interactive terminal, filesystem writes, safe config edits, lifecycle, and cleanup — all behind a uniform safety model.

**Architecture:** Three tiers extend the existing read path. The **contract** (zod) gains write verbs + two hub→node frames (`input`/`resize`) + `StreamMsg.enc`. The **node-agent** (Go) gains a Go-native PTY `SessionManager` (ring-buffer scrollback, detach/reattach) and path-jailed fs/lifecycle/cleanup handlers. The **hub** (Bun/Hono) gains a console↔hub terminal **WebSocket** bridge, REST write routes (capability-gated + audit-logged), and a `station_audit` table. The **console** (Svelte 5) gains an xterm.js terminal panel, a writable file browser + config diff editor, lifecycle/cleanup controls, confirm dialogs, and an Activity view.

**Tech Stack:** TypeScript + zod (contract); Go + `github.com/creack/pty` (node-agent); Bun + Hono + Drizzle + Better Auth (hub); SvelteKit + Svelte 5 runes + `@xterm/xterm` + `@xterm/addon-fit` (console).

**Spec:** `docs/superpowers/specs/2026-06-28-p2-write-ops-design.md`.

## Global Constraints

- **Branch:** `redesign/fleet-console`. Frequent commits (one per task min).
- **Safety is mandatory, four layers:** path-jail every `fs.*` op to the station `workspacePath` at the NODE (reject `..`, symlink escape, absolute-outside); capability-gate every write verb at the HUB (reject if the station didn't advertise the capability); write a `station_audit` row at the HUB before dispatching every mutation; tiered confirm at the CONSOLE (plain for backup'd writes, type-to-confirm for delete/cleanup/lifecycle-stop). Terminal keystrokes are NEVER logged.
- **Terminal = fresh shell** (`$SHELL`, fallback `/bin/sh`, `cwd=workspacePath`); NOT attached to any running agent process. **One terminal per station** (protocol still carries `sessionId`).
- **Durability:** the PTY lives inside the long-running node-agent — survives console reload / network blip / hub restart; does NOT survive node-agent restart (tmux backend is a future, non-goal here).
- **Tests:** Go `go test ./...` (node-agent); vitest (hub + console). Hub integration suites MUST call the existing `ensurePgMigrations()` helper and use the local docker postgres on `:5434` (user/pw `agentpod`/`agentpod-dev-password`) — NEVER the SSH-tunnel DBs on `:5432`/`:5433`.
- **Encoding:** terminal stdin/stdout are binary → base64 over the wire (`input.data` and `StreamMsg{enc:"base64"}`); `logs.tail` stays utf8.
- **Existing patterns:** node verbs are dispatched via the `Handler` interface in `apps/node-agent/internal/gateway/dispatch.go`; descriptors register in `apps/node-agent/cmd/agentpod-node/registry.go`; hub station routes live in `apps/hub/src/routes/stations.ts` and call the broker in `apps/hub/src/services/broker.ts`; console station panels live under `apps/console/src/lib/components/stations/` and the station page `apps/console/src/routes/nodes/[id]/stations/[stationId]/+page.svelte`.

```go
func TestOpenAttachReplayAndDetach(t *testing.T) {
    m := NewManager()
    defer m.Shutdown()
    // /bin/cat echoes stdin to stdout — deterministic, no shell needed.
    s, err := m.Open("station-1", "/bin/cat", t.TempDir(), 80, 24)
    if err != nil { t.Fatal(err) }

    ch, unsub := s.Subscribe()
    if err := s.Write([]byte("ping\n")); err != nil { t.Fatal(err) }
    got := readUntil(t, ch, "ping")   // helper: drain channel until substring or timeout
    if !strings.Contains(got, "ping") { t.Fatalf("want echo, got %q", got) }
    unsub() // detach — session must stay alive

    // Re-open by key returns the SAME session (one-per-station)
    s2, _ := m.Open("station-1", "/bin/cat", t.TempDir(), 80, 24)
    if s2.ID != s.ID { t.Fatalf("expected same session, got %s vs %s", s2.ID, s.ID) }

    // Re-subscribe replays scrollback (the earlier "ping" echo)
    ch2, unsub2 := s2.Subscribe(); defer unsub2()
    replay := readUntil(t, ch2, "ping")
    if !strings.Contains(replay, "ping") { t.Fatalf("expected scrollback replay, got %q", replay) }
}
```
