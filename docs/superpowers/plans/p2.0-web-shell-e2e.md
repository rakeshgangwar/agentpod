# P2.0 — Console Web-Shell E2E Runbook (verified)

Verified the web console end-to-end in a real browser (Playwright) on this laptop,
against the real hub + a real node-agent. The shell port (off Tauri) makes the new
station pages reachable in a browser.

## Stack
```bash
# DB (local docker postgres on :5434)
docker exec agentpod-test-postgres psql -U agentpod -d agentpod -c 'DROP DATABASE IF EXISTS uidemo;' -c 'CREATE DATABASE uidemo;'
docker exec agentpod-test-postgres psql -U agentpod -d uidemo -c 'CREATE EXTENSION IF NOT EXISTS vector;'
# hub on :3001 (self-migrates on start)
( cd apps/hub && DATABASE_URL=postgres://agentpod:agentpod-dev-password@localhost:5434/uidemo PORT=3001 bun run start & )
# console on :1420 pointed at the hub
( cd apps/console && PUBLIC_HUB_URL=http://localhost:3001 pnpm dev & )
# node-agent
( cd apps/node-agent && go build -o /tmp/agentpod-node ./cmd/agentpod-node )
```

## Verified browser flow
1. Open `http://localhost:1420` → "connect to API" screen (prefilled `http://localhost:3001`) → **Connect** — succeeds via the **web fetch** path (`GET /health`), NOT Tauri. ✅
2. **Create account** (email/password, Better Auth) → redirected into the app, authenticated via the **session cookie** (no Tauri). ✅
3. `/nodes` → `GET /api/nodes` succeeds with the cookie. Create an **enrollment token** in the UI.
4. `agentpod-node enroll --hub http://localhost:3001 --token <token>` then `agentpod-node run &` → node shows **online** in `/nodes`. ✅
5. Open the node → **Detected Stations**: ~20 real Claude Code projects + OpenClaw, each with **Adopt**. ✅
6. **Adopt** a station (`agentpod`) → appears under **Adopted Stations** → open it:
   - **Health**: real data (Disk 2.99 GB, Last Activity timestamp, Status Stopped). ✅
   - **Files**: read-only browse of the real project dir + file preview. ✅
   - **Logs**: SSE tail. ✅ (see findings)

## Findings (from the visual E2E)
- **FIXED in this phase:** the station **Logs** SSE returned **401** — `EventSource` doesn't send cookies cross-origin by default. Fixed with `new EventSource(url, { withCredentials: true })` (commit on `redesign/fleet-console`).
- **FOLLOW-UP (tracked):** the Claude descriptor's `logs.tail` streams the **entire** session `.jsonl` from the start (a single session can be ~18 MB). With `withCredentials` working, the full stream floods the browser. `logs.tail` should **tail the last N lines/bytes** (node-side) and the `LogTail` UI should **cap rendered lines**.

## Teardown
```bash
kill $(lsof -ti:3001) $(lsof -ti:1420); pkill -f agentpod-node
docker exec agentpod-test-postgres psql -U agentpod -d agentpod -c 'DROP DATABASE IF EXISTS uidemo;'
```

The legacy OpenCode pages (`/projects`, `/workflows`, `/admin`, `/settings`) still use Tauri and are non-functional in the browser — out of scope (being retired). Only the shell + the `/nodes` station flow are web-functional.
