# Phase 4: Mobile App Core Features

Implement the main user-facing features: project management, chat interface, file browser, and real-time updates.

## Objectives

1. Project list and creation UI
2. OpenCode chat interface with streaming
3. File browser with syntax highlighting
4. Real-time updates via SSE
5. Container status management

## Prerequisites

- Phase 2 complete (Management API running)
- Phase 3 complete (Mobile foundation working)

## Duration

**Estimated:** 5-7 days

## Deliverables

- [ ] Project list with status indicators
- [ ] Create project flow (empty + GitHub import)
- [ ] Chat interface with real-time streaming
- [ ] File browser and viewer
- [ ] Start/stop project controls

## Success Criteria

1. Can create a project from the app
2. Can send prompts and see streamed responses
3. Can browse files and view with syntax highlighting
4. Status updates appear in real-time
