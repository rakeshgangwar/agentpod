# CodeOpen JavaScript Flavor

JavaScript and TypeScript development environment.

## Languages

- JavaScript (ES2024)
- TypeScript

## Included Tools

### Package Managers
- npm (via Node.js)
- pnpm
- yarn
- bun

### Build Tools
- esbuild
- vite
- turbo
- tsx

### Development Tools
- typescript
- ts-node
- biome
- prettier
- eslint
- npm-check-updates

### Runtimes
- Node.js 22 LTS
- Deno (latest)
- Bun (latest)

## Frameworks Supported

- React
- Vue
- Svelte
- Next.js
- Nuxt
- Express
- Fastify
- Hono
- Nest.js

## Usage

```bash
docker run -it codeopen-js:latest
```

## Building

```bash
./docker/scripts/build-flavor.sh js
```
